Назначение

Данная программа представляет из себя блокнот-редактор кода с поддержкой семантики языков программирования (список поддерживаемых языков будет расширяться по мере развития проекта). Цель проекта — освоить основные функциональные возможности системы контроля версий в ходе работы над проектом.

Программные модули

Редактор кода

  1. Отвечает за основной интерфейс программы;
  2. Обеспечивает редактирование кода;
  3. Обеспечивает подсветку синтаксиса;
  4. Поддерживает функцию автоформатирования кода;
  5. Поддерживает функцию автозавершения слов.

Окно терминала

  1. Отвечает за интеграцию терминала;
  2. Поддерживает разные коммандные оболочки.

Анализатор семантики языков

  1. Анализирует код и разбивает его на семантические блоки;
  2. Выполняет проверку структуры кода.

Модуль поиска и замены

  1. Инструмент поиска текста в коде;
  2. Есть функция подсчёта вхождений;
  3. Есть функция поиска по регулярному выражению;
  4. Поддерживает замену найденных вхождений на заданный текст.

Модуль управления темами

  1. Отвечает за изменение внешнего вида интерфейса при помощи тем оформления;
  2. Позволяет загружать новые темы;
  3. Позволяет настраивать темы.

Модуль настройки программы

  1. Содержит основные настройки программы;
  2. Позволяет настраивать работу других модулей системы.
